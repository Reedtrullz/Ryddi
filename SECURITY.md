# Security Policy

Ryddi is a local-first disk space manager. Security reports are welcome.

## Reporting

Use GitHub Security Advisories: https://github.com/Reedtrullz/Ryddi/security/advisories/new

## Trust Boundaries

Ryddi should not:
- Upload paths, reports, scans, or command output
- Store SSH private keys, passwords, or API keys
- Run destructive commands without user confirmation
- Treat browser profiles, VM/container disks, credentials, user documents, photos, music projects, app state databases, Codex/Hermes sessions/memories/config/auth, or unknown user data as auto-safe

## Privacy

No telemetry, no uploads, no remote analysis. Everything stays on your Mac.
