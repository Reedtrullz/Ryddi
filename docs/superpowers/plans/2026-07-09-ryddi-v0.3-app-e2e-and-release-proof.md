## App UX Requirements

- Summary has one primary action.
- Sidebar labels do not clip at minimum supported width.
- Toolbar controls wrap or collapse into a menu at small width.
- Empty states explain the current state and offer an action.
- Permission/access UI provides:
  - Open Full Disk Access settings
  - Copy app bundle path
  - Recheck Access
  - Show inaccessible configured scopes
  - Explain that restarting the app may be required after Full Disk Access changes
- Review queues support list, filter, detail, and plan creation.
- Reclaim button remains disabled with an explicit reason until dry-run state is current.
- Remote Targets show probe/scan/export only; no reclaim.
